# PK道場 ⚽️🍁
「スポーツの秋」をテーマにした
PK ゲームアプリです。
君は何連ゴールできるかな？🔥

## 🎮 遊び方
1. ログインして「PK挑戦」ページへ
2. 左・中・右からシュート方向を選択
3. キーパーと読み合い！⚽️🧤
4. 結果が履歴・戦績に自動で保存されます

## 📊 戦績機能
- ゴール数 / セーブ数 / 成功率を記録
- 連続ゴール数をカウント
- 履歴一覧ページで詳細確認

## 🌟 特徴
- シンプルな 3 択ゲーム
- キーパーが動いてセーブ演出
- 戦績サマリ + 一覧ページ
- 季節演出 🍁 紅葉が舞い落ちる（秋仕様）

## 🚀 デプロイ
- Render にて稼働予定
- Ruby on Rails 7 / PostgreSQL / Docker

## 📝 今後の拡張
- 対戦機能（リアルタイム PK バトル）
- ランキング機能
- 季節ごとの演出（冬: ❄️, 春: 🌸, 夏: ☀️）
